# ⚡ TechForge - Portfólio & Landing Page (V1)

**TechForge** é uma aplicação web moderna de alto desempenho desenvolvida como a V1 do meu portfólio profissional e landing page de serviços. 

Este projeto foi construído para servir a um **propósito duplo**:
1. 🔧 **Técnico de Informática:** Uma vitrine elegante para captação de clientes, exibindo serviços de manutenção, montagem de setups gamers, reparos em placas-mãe e gestão de hardware.
2. 💻 **Desenvolvedor Frontend:** A prova viva das minhas habilidades de desenvolvimento de software, arquitetura de UI/UX, e fluência com o ecossistema moderno do React.

---

## 🌟 Destaques e Funcionalidades

- **UI/UX Premium e Moderna:** Design focado em conversão e experiência de usuário, utilizando uma paleta Dark Mode fluida com detalhes em "Brand Red".
- **Interações e Animações Complexas:** 
  - Cards interativos com física em 3D e efeitos de inclinação (Tilt) baseados na posição do mouse.
  - "Glow effects" (brilhos radiais) acompanhando o cursor para um aspecto "Gamer/Tech" premium.
  - Galerias automáticas com indicadores de progresso dinâmicos estilo Steam.
- **Desenvolvimento Componentizado:** Código limpo, componentização rigorosa (Hero, About, Services, Contact, Reviews) e tipagem segura.
- **Totalmente Responsivo (Mobile-First):** Layout perfeitamente adaptado para todos os tamanhos de tela.
- **Integração Realista:** Layout de seções para Avaliações (Google Reviews simulados/integrados) e Contatos via WhatsApp / Mapa Local.

## 🛠️ Tecnologias Utilizadas

Este projeto foi concebido utilizando o que há de mais rápido e eficiente no mercado de frontend:

- **React 19** - Biblioteca base para a construção das interfaces.
- **TypeScript** - Segurança e inteligência no código com tipagem estática rigorosa.
- **Vite** - Bundler de altíssima velocidade para uma experiência de desenvolvimento instantânea.
- **Tailwind CSS 4** - Estilização utilitária super-rápida, permitindo micro-animações, gradientes e controle de responsividade sem sair do JSX.
- **Framer Motion** - Responsável pelas animações de entrada fluidas (fade-ins e transições de tela).
- **Lucide React** - Conjunto de ícones consistentes, leves e de código aberto.
- **React Helmet Async** - Gerenciamento otimizado de SEO, Title Tags e Meta-descriptions dinâmicas.

## 🚀 Como Executar Localmente

Siga as instruções abaixo para rodar o projeto em sua máquina local.

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/techforge-site.git

# Entre no diretório do projeto
cd techforge-site

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento Vite
npm run dev
```

Abra [http://localhost:5173](http://localhost:5173) no seu navegador para ver o projeto rodando.

## 📁 Estrutura de Diretórios (Resumo)

- `src/components/` - Contém as fatias da landing page (Navbar, Hero, About, Gallery, Reviews, Contact, Footer).
- `src/App.tsx` - O entrypoint da UI, onde todas as seções são renderizadas em bloco com navegação fluida.
- `index.html` - Base do projeto carregando fontes do Google e metadados.
